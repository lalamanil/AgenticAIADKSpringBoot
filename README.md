# AgenticAIADKSpringBoot
This repository demonstrates LLM-powered Agentic AI applications integrated with Spring AI MCP(Model context Protocol) tools. It includes an SSE listener subscribing to the /sse endpoint for asynchronously published MCP events and performs request-response correlation using JSON-RPC message IDs.

Please refer to AgenticAIADKImplementation.pdf and ArchitectureDiagram.pdf for a detailed implementation guide, architecture overview, and end-to-end execution flow of the Google ADK Agent integrated with Gemini, Vertex AI, and MCP tools.
